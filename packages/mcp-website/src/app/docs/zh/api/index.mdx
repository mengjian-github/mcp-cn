> MCP API 完整参考文档

欢迎使用 MCP API！本文档提供了 MCP API 的完整参考信息。

## CLI 工具

MCP 提供了功能强大的命令行工具，可以通过 NPM 安装：

```bash
npm install @mcp_hub/cli
```

### 命令列表

#### `mcp install`

安装指定的服务器。

```bash
mcp install <server-id> --client <client-name> [--env <config>] [--key <api-key>] [--ppe]
```

参数说明：
- `server-id`：必需，要安装的服务器ID
- `--client`：必需，指定客户端，可选值：`cline`、`cursor`、`claude`
- `--env`：可选，JSON格式的配置信息
- `--key`：可选，API密钥
- `--ppe`：可选，启用PPE模式

#### `mcp uninstall`

卸载指定的服务器。

```bash
mcp uninstall <server-id> --client <client-name>
```

参数说明：
- `server-id`：必需，要卸载的服务器ID
- `--client`：必需，指定客户端

#### `mcp run`

运行指定的服务器。

```bash
mcp run <server-id> [--env <config>] [--key <api-key>] [--ppe]
```

参数说明：
- `server-id`：必需，要运行的服务器ID
- `--env`：可选，JSON格式的配置信息
- `--key`：可选，API密钥
- `--ppe`：可选，启用PPE模式

#### `mcp list`

列出可用的服务器。

```bash
mcp list [category]
```

参数说明：
- `category`：可选，按类别筛选服务器

#### `mcp inspect`

检查服务器的详细信息。

```bash
mcp inspect <server-id>
```

参数说明：
- `server-id`：必需，要检查的服务器ID

### 全局选项

所有命令都支持以下选项：

- `--verbose`：显示详细的调试信息
- `--help`：显示帮助信息
