## 💡 什么是 MCP？

MCP（Model Context Protocol）是一个开放标准协议，它是 AI 世界的"USB-C 接口"，是 AI 🧠 连接外部工具的神经元。MCP协议有以下特性：
- 为大型语言模型（LLM）应用提供标准化接口
- 让 AI 系统能够安全、便捷地连接并调用外部数据源与工具
- 统一不同 AI 平台的工具调用体验  

你可通过各类支持 MCP 协议的 AI 客户端（**MCP Client**）使用这些外部数据源和工具（**MCP Server**）

> 查看 <a href="https://modelcontextprotocol.io/introduction" target="_blank">基础概念</a> 了解更多 MCP 相关介绍

## 🌐 使用 MCP Server

### 🎯 第一步：选择 AI 客户端

多个主流 AI 客户端支持了 MCP 协议，包括不限于：

- <a href="https://www.trae.com.cn/" target="_blank">Trae CN</a>：Trae 旗下的 AI IDE 产品，无缝融入你的开发流程
- <a href="https://www.cursor.com/" target="_blank">Cursor</a> - AI 驱动的智能代码编辑器
- <a href="https://cline.bot" target="_blank">Cline</a> - 适用于 VS Code 的 AI 编码插件
- <a href="https://windsurf.com/" target="_blank">windsurf</a> - 一款 "更适合AI编程新手" 的AI编程工具

> 查看 <a href="/docs/clients" target="_blank">客户端指南</a> 了解各个平台的详细配置和使用方法。

### ⚙️ 第二步：安装 MCP Server

MCP Hub 平台提供两种安装 MCP Server 的方式，使用前请先在你设备上安装 <a href="https://nodejs.org/zh-cn" target="_blank" >`Node.js`</a>。以下以 `Cline` 为例介绍安装 `sequential-thinking` MCP Server 的流程


####  💻 通过 CLI 安装(推荐)

1. 访问 MCP Hub <a href="/" target="_blank">首页</a>， 在MCP 服务市场中筛选你感兴趣的 MCP Server。
2. 点击卡片，在详情页找到和你的 AI 客户端对应的 NPM 安装指令
```bash
npx -y @mcp_hub/cli@latest install 
sequential-thinking --client cline
```
3. 复制指令，在任意终端中执行上述指令。

#### 📝 通过 JSON 配置安装

1. 访问 MCP Hub <a href="/" target="_blank">首页</a>， 在 MCP 服务市场中筛选你需要的 MCP Server。
2. 在详情页找到和你 AI 客户端对应的 JSON 配置
```bash
{
  "mcpServers": {
    "sequential-thinking": {
      "command": "npx",
      "args": [
        "-y",
        "@mcp_hub/cli@latest",
        "run",
        "sequential-thinking"
      ]
    }
  }
}
```
3. 复制 JSON 配置，将该配置粘贴到你 AI 客户端的 MCP 配置文件中（查找配置文件可见 <a href="/docs/clients" target="_blank">客户端指南</a>）。

### 🎮 第三步：开始使用

在 AI 客户端的对话框中输入`用sequential-thinking来深入介绍一下 MCP Server 的使用技巧`，开启你的 MCP 之旅 ！
