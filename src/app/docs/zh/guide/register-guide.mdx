> 在 MCP Hub 平台上注册 MCP Server 非常简单，请按照以下步骤操作

## 🌟 注册流程

### 1. 打开注册面板

点击平台官网右上角 **新增 Server** 按钮，打开注册面板。

### 2. 填写服务参数

按照页面提示填写服务参数，相关参数说明如下：

- 📝 **服务名称**：平台上展示的 MCP Server 名称
- 📦 **BNPM 包名**：将自己的 MCP Server 发布到 NPM 后，填入包名
- 📄 **描述（可选）**：介绍 MCP Server 的核心功能和使用场景，默认读取 BNPM 包中的描述
- ⚙️ **MCP 服务配置**：平台会根据包名自动生成配置；如需注册环境变量，需要在 **env** 参数中指定

```json
{
  "<mcp-server-name>": {
    "command": "npx",
    "args": [
      "-y",
      "<bnpm-package-name>"
    ],
    "env": {
      "API_KEY": "<your_api_key>",
    }
  }
}
```

### 3. 提交参数完成部署

点击 `确定` 按钮提交参数，在 MCP Server 市场中可找到你的 Server，分享伙伴一起来使用吧！ 🎉
